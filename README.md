
# Free HTML Email Templates
Free HTML Email templates created using the [Postcards Email Builder](https://designmodo.com/postcards/).

**Features**
*	Drag & drop builder
*	Edit text and images live
*	Upload or use External images
*	Preview the result
*	Unlimited exports
*	Direct email preview
*	Responsive
